# PickMeUp v1.4.1

发布日期：2026-07-11

## 安装包

- **Android APK**：`PickMeUp-1.4.1-release.apk`（见本 Release 附件）
- 应用 ID：`com.pickmeup.assistant`
- `versionName`：`1.4.1` / `versionCode`：`5`

安装说明：下载 APK 后在 Android 设备上允许「安装未知应用」，直接安装即可。若已安装旧版本且签名不同，需先卸载再装。

## 亮点

- 大幅优化列表与交互流畅度，降低 Android WebView 卡顿
- 减少毛玻璃与布局动画带来的合成开销，滚动与点击更跟手
- 搜索、存储写入与短信批量导入改为防抖 / 分片，避免主线程长时间阻塞

## 变更内容

### 性能

- 包裹卡片移除 `layout` 弹簧动画，改用轻量进出场；工具栏按钮改 CSS `active-scale`
- 列表卡片取消 `backdrop-filter`，仅顶栏保留轻量毛玻璃
- `PackageCard` / 工具栏 / 热力图 `memo`，事件回调 `useCallback` 稳定引用
- 卡片启用 `content-visibility: auto` 与 CSS containment，降低屏外绘制成本
- `localStorage` 写入防抖；页面隐藏时 flush，避免丢数据
- 搜索输入防抖 + `startTransition`，避免每键触发完整列表重算
- 短信自动 / 批量导入使用同步提取，并每 8 条让出主线程
- 重复取件码短路时不再无意义 `setState`

### 版本

- 版本号升级至 `1.4.1`（Android `versionCode=5`）

## 已知说明

- 短信读取与自动导入仍仅在 Android 原生环境可用
- 删除 / 已取件记录在本地保留 3 天防重，过期后可再次导入相同取件码

## 建议验证

1. 安装 APK，列表滚动、切换待取/已取、标记已取件是否流畅
2. 搜索栏连续输入是否仍跟手
3. 批量导入多条短信时界面是否可继续操作
4. 杀进程后数据是否正常保存
