# PickMeUp v1.3.0

发布日期：2026-03-19

## 亮点

- 优化图片 OCR 识别链路，提升取件码截图导入的成功率和错误可读性
- 完成 Android release 打包配置整理，仓库不再保存签名敏感信息
- 补齐公开仓库所需的 README、License 和版本说明

## 变更内容

- 为 OCR 增加图片预处理，提升低质量截图的识别效果
- 强化规则兜底与错误透传，识别失败时能给出更明确的提示
- 版本号升级至 `1.3.0`，Android `versionCode=3`
- release 签名改为从本地 `keystore.properties` 或环境变量读取

## 已知说明

- 短信读取与自动导入功能仅在 Android 原生环境可用
- GitHub Release 默认提供源码归档；如需附加 APK，请先在本地完成签名构建
